### 论文概述
#### 研究背景
旧石器时代中晚期过渡阶段（5–3万年前），伊比利亚半岛尼安德特人被现代人取代。MIS 3期气候冷暖波动被认为是关键影响因素，但气候如何通过生态系统影响尼安德特人消失的时空规律尚不明确。

#### 研究方法
1. 采用贝叶斯、最优线性估计、累积概率分布重建文化层年代
2. 利用动态植被模型（LPJ-GUESS）估算净初级生产力（NPP）
3. 结合宏生态模型与现代观测数据，计算食草动物承载量
4. 对比欧洲西伯利亚区、地中海区等4个生物地理区的差异

#### 研究内容
量化MIS 3期冷暖期对植物与食草动物生物量的影响，检验生态系统生产力是否决定尼安德特人消失与现代人到来的时空模式。

#### 核心结论
1. 北部欧洲西伯利亚区：尼安德特人消失与NPP、食草动物生物量骤降同步，现代人到来与资源回升吻合
2. 南部地中海区：冷暖期生产力波动小、中型食草动物生物量更稳定，尼安德特人存续更久
3. 生态生产力波动是伊比利亚尼安德特人南北差异消失的关键原因，南部成为气候避难所
   
#### 研究结果与内容解读

一、全文研究逻辑（从问题到结论）
1. 科学问题：气候波动如何通过生态生产力→食草动物生物量的 bottom-up 效应，影响尼安德特人消亡与现代人替代
2. 研究设计：
   - 年代框架：贝叶斯模型+最优线性估计+概率叠加，重建旧石器时代过渡时间线
   - 生态建模：LPJ‑GUESS模型估算净初级生产力（NPP）→宏生态模型估算食草动物承载力
   - 时空耦合：对比四大生物地理区NPP、食草动物生物量与莫斯特文化（尼人）、奥瑞纳文化（现代人）的时空分布
3. 核心结论：生态生产力稳定性决定尼安德特人存续时间，南部高稳定区成为最后避难所。

 二、核心图表精准解读
1. 图1（基于Table1绘制：NPP空间格局与波动）
- 图表信息：伊比利亚四大生物地理区冰期/间冰期NPP均值、标准差、变异系数、统计显著性
- 关键数据：
  - 间冰期：欧洲西伯利亚区NPP最高（0.339 kg km⁻² yr⁻¹），整体北高南低
  - 冰期：仅热地中海区NPP无显著下降（p=0.986），欧洲西伯利亚+上地中海区骤降、变异系数翻倍
- 核心解读：
  1.  南北梯度：北部生产力高但极不稳定，南部低但抗冰期扰动能力极强
  2.  波动差异：北部NPP冰期降幅≈0.06，南部仅≈0.01，是尼人“北退南存”的生态基础
  3.  统计支撑：威尔科克森检验显示北部冰期/间冰期差异极显著（p<10⁻¹⁴），南部无差异，结论稳健。

2. 图3（不同体型食草动物承载力：冰期vs间冰期）
- 图表信息：四大区域大/中-大/中/小型食草动物承载力箱线图，分冰期/间冰期
- 关键发现：
  - 大型食草动物：欧洲西伯利亚区始终最高，热地中海区最低
  - 小型食草动物：格局完全相反，热地中海区最高
  - 中型/中-大型（尼人核心猎物）：
    - 间冰期：北部略高
      -冰期：北部骤降≈1/3，南部仅降3%–8%，南部反超北部
  核心解读：
  1. 猎物适配性：尼人主食中型/中-大型食草动物，冰期北部食物断崖式短缺，南部稳定供给
  2. 避难所机制：南部不是“贫瘠退路”，而是冰期最优觅食区，完美解释尼人长期滞留
  3. 生态位分化：南北猎物结构差异，决定人群适应策略与存续能力。

3. 图5（MIS3期食草动物承载力+文化序列时空耦合）
图表信息：四大区域不同体型食草动物承载力时序变化+莫斯特/夏特佩赫/奥瑞纳文化存续时段
  关键耦合：
  1.  欧洲西伯利亚区：GS‑12冰期NPP↓33.3%、食草动物↓45%，莫斯特同步消失；GI‑11生产力回升，奥瑞纳出现
  2.  上地中海区：GS‑11/10生产力连续暴跌，莫斯特消失，无现代人长期占据
  3.  中/热地中海区：生产力波动极小，莫斯特最晚消失（热地中海区≈3.5–3.3万年前），与奥瑞纳重叠最久
  核心解读：
  1.  时间锁相：尼人消亡时间≈区域生产力崩溃时间，现代人到来≈生产力恢复时间
  2.  空间分异：波动越强→消失越早；稳定→存续越久
  3.  机制闭环：气候→NPP→食草动物→人类承载力→文化更替，完整因果链。


三、三张图的整体科学价值（串联成核心证据链）
1. 图1（生产力基底）：划定南北生态稳定性边界，奠定时空分异的物理基础
2. 图3（资源供给）：锁定中型食草动物为关键限制因子，揭示南部冰期资源优势
3. 图5（时空耦合）：把生产力波动、猎物丰度、文化序列精准对齐，直接证明生态驱动假说

三者共同证明：伊比利亚尼安德特人消失不是单一竞争或气候事件，而是生态系统承载力时空波动驱动的适应性衰退与替代过程。

本文核心结论：IS3期（5–3万年前）冰期-间冰期旋回引发的生态系统生产力波动，是伊比利亚半岛尼安德特人时空消失的关键生态驱动因子；北部欧洲西伯利亚区生产力骤降导致尼安德特人早消失，南部地中海区生产力稳定成为其气候避难所，最终形成“北早南晚”的时空消亡格局。
