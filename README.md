### 论文概述
#### 研究背景
旧石器时代中晚期过渡阶段（5–3万年前），伊比利亚半岛尼安德特人被现代人取代。MIS 3期气候冷暖波动被认为是关键影响因素，但气候如何通过生态系统影响尼安德特人消失的时空规律尚不明确。

#### 研究方法
1. 采用贝叶斯、最优线性估计、累积概率分布重建文化层年代
2. 利用动态植被模型（LPJ-GUESS）估算净初级生产力（NPP）
3. 结合宏生态模型与现代观测数据，计算食草动物承载量
4. 对比欧洲西伯利亚区、地中海区等4个生物地理区的差异

#### 研究内容
量化MIS 3期冷暖期对植物与食草动物生物量的影响，检验生态系统生产力是否决定尼安德特人消失与现代人到来的时空模式。

#### 核心结论
1. 北部欧洲西伯利亚区：尼安德特人消失与NPP、食草动物生物量骤降同步，现代人到来与资源回升吻合
2. 南部地中海区：冷暖期生产力波动小、中型食草动物生物量更稳定，尼安德特人存续更久
3. 生态生产力波动是伊比利亚尼安德特人南北差异消失的关键原因，南部成为气候避难所
   
#### 研究结果与内容解读
一、研究背景与问题

尼安德特人如何在约5万至3万年前被现代人类（AMH）取代，是古人类学的重要谜题。本文聚焦于伊比利亚半岛，该地区是尼安德特人最后生存的区域之一。研究试图回答：生态系统生产力（尤其是植物和食草动物生物量的波动）是否以及如何影响了尼安德特人的消失与现代人类的扩散？

 二、研究方法简述

研究整合了三种模型方法：
1. 年代学重建：使用贝叶斯年代模型、最优线性估计（OLE）和考古组合的核密度分布（SPD），更新了伊比利亚四个生物地理区（欧亚区、高地中海区、中地中海区、热地中海区）的旧石器时代中期-晚期过渡年代框架。
2. 净初级生产力（NPP）模拟：利用动态植被模型LPJ-GUESS，结合古气候模拟数据，估算55–30 kyr BP期间每个遗址周围NPP的演变。
3. 食草动物承载力估算：基于现代生态学数据建立宏观生态模型，估算不同时期、不同区域各体型食草动物的生物量。

 三、主要结果解读

 1. 不同区域生产力变化差异显著
欧亚区（北部） 和 高地中海区（中部内陆 在寒冷期（stadial）NPP下降剧烈（分别减少约18%和25%），且波动幅度大。
中地中海区和热地中海区（南部）的NPP变化平稳，寒冷期下降不足5%，表现出气候缓冲效应。
热地中海区在寒冷期甚至保持了与温暖期相当的NPP水平。

2. 食草动物生物量变化与人类文化更替同步
欧亚区：莫斯特文化（尼安德特人）在GS-12寒冷期消失，与此同时NPP下降33%，中大型食草动物生物量下降45%。随后在GI-11回暖期，夏特佩龙文化和奥瑞纳文化（可能为现代人或晚期间尼安德特人）迅速出现。
中地中海区：莫斯特与奥瑞纳文化有至少400年重叠，NPP和食草动物生物量波动仅约8%，环境相对稳定。
热地中海区：莫斯特文化持续到约35–33 kyr BP，与奥瑞纳文化重叠时间最长，且寒冷期中-大型食草动物生物量最高。

3. 支持“生态驱动”假说
北方尼安德特人的消失与食物资源锐减时间吻合，支持气候通过影响生产力间接导致人口衰退。
南方地区生产力稳定、猎物丰富，成为尼安德特人的避难所，推迟了他们的消失。

 4. 对“埃布罗河屏障”假说的修正
传统观点认为埃布罗河谷是尼安德特人与现代人之间的地理屏障。本研究发现，真正的生态屏障可能位于伊比利亚中部内陆（高地中海区），那里生产力极低且波动剧烈，不利于任何人类群体长期生存。而沿海地区（包括地中海和大西洋沿岸）生产力一直较高，可能促进了现代人的快速扩散。

5. 竞争假说的再审视
在北方，如果夏特佩龙文化仍属尼安德特人，则可能与现代人存在资源竞争；但在南方，长时间共存且资源稳定，竞争可能并不激烈。因此，竞争不能作为单一解释，不同区域应有不同机制。

四、研究意义与局限

意义：
 首次将NPP、食草动物承载力与考古年代模型结合，量化了气候波动通过食物网传递对人类种群的影响。
揭示了伊比利亚半岛内部生态异质性如何塑造了尼安德特人消失的时空模式。
为理解末次冰期气候事件与人类演化的耦合关系提供了新证据。

局限：
年代学数据仍存在不确定性（尤其夏特佩龙文化样本少）。
SPD作为人口代理指标需谨慎使用。
古气候模拟与NPP估算存在模型偏差，但敏感性分析表明主要结论稳健。
未考虑捕食压力、疾病、文化竞争等非资源因素。

五、结论

伊比利亚半岛尼安德特人的消失并非单一原因所致，而是气候驱动的生态系统生产力波动在不同区域以不同方式作用的结果。北方资源锐减直接冲击了尼安德特人的生存，而南方稳定的生产力提供了长期避难所。现代人的扩散则利用了沿海高生产力走廊。这一发现强调了生态学视角在人类演化研究中的核心地位。
